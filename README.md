# Bayesian-data-analysis-Project
A Bayesian project on the Reisby dataset to see if the drug imipramine is effecting depression levels. R jags will be needed to run this.

This data is from a psychiatric study conducted by Reisby et al.(1977)1. The study investigated response of patients to the antidpressant drug imipramine(IMI). Patients received 225mg/day dosesof IMI for 4 weeks.

The report contains the snipets of code needed to run each model
